Detailed Description

template<class T>
class microgradpp::_Tensor2D< T >

A class representing a 2D tensor (matrix) of ValuePtr objects.

This class provides methods for tensor operations, including zeroing gradients, accessing elements, and pushing back new rows of tensors.

Template Parameters
TThe type of tensor (e.g., std::vector<ValuePtr>).

template<class T >
void microgradpp::_Tensor2D< T >::zeroGrad ( )
inline

Zeros the gradients of all elements in the 2D tensor.

This method iterates through each sub-tensor (row) and sets the gradient of each ValuePtr to zero. This is typically used before starting a new forward/backward pass in training a neural network to ensure gradients from the previous pass do not affect the current calculations.
