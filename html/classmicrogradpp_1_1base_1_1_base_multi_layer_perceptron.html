Detailed Description

Abstract base class for Multi-Layer Perceptron (MLP) networks.

This class provides a foundation for implementing multi-layer perceptron models. It encapsulates a Sequential object, which contains a series of neural network layers, and provides methods to print parameters, reset gradients, and update parameters. Derived classes must implement the forward method to define the forward pass logic.
