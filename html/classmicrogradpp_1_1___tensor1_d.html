Detailed Description

template<class T>
class microgradpp::_Tensor1D< T >

A class representing a 1D tensor (vector) of ValuePtr objects.

This class provides methods for tensor operations, including zeroing gradients, accessing elements, and pushing back new values.

Template Parameters
TThe type of tensor (e.g., std::vector<ValuePtr>).
