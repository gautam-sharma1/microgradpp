\doxysubsection{Detailed Description}
Manages a sequence of neural network layers and supports forward propagation. 

The {\ttfamily \doxylink{classmicrogradpp_1_1core_1_1_sequential}{Sequential}} class allows users to stack neural network layers in a defined sequence, providing functionality for forward propagation, parameter access, gradient updates, and zeroing. This class is essential for constructing feedforward neural networks in the microgradpp library.
