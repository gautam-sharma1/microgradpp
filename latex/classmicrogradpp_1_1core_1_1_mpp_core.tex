\doxysubsection{Detailed Description}
Abstract base class for core components of neural network layers. 

The {\ttfamily \doxylink{classmicrogradpp_1_1core_1_1_mpp_core}{Mpp\+Core}} class serves as an interface for all core neural network layers. It defines essential functions for neural network layers, including parameter management, gradient resetting, and forward computation, which must be implemented by any class that inherits from {\ttfamily \doxylink{classmicrogradpp_1_1core_1_1_mpp_core}{Mpp\+Core}}.
