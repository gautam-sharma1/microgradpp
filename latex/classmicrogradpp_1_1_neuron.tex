\doxysubsection{Detailed Description}
Represents a single neuron in a neural network. 

The \doxylink{classmicrogradpp_1_1_neuron}{Neuron} class contains weights and a bias that are used for calculating the output of the neuron given an input tensor. It also provides methods for parameter management, gradient resetting, and output computation.
