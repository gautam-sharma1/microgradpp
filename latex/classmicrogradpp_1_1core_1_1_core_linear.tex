\doxysubsection{Detailed Description}
Represents a linear (fully connected) layer with configurable input and output dimensions. 

The {\ttfamily \doxylink{classmicrogradpp_1_1core_1_1_core_linear}{Core\+Linear}} class is derived from {\ttfamily \doxylink{classmicrogradpp_1_1core_1_1_mpp_core}{Mpp\+Core}} and implements the forward operation, zero gradient function, and parameter retrieval for a linear neural network layer.
